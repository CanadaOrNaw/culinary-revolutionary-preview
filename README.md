# Culinary Revolutionary — redesign concept

A dependency-free, responsive static prototype built from the business's publicly served logo, photography, and published content.

## Preview locally

```bash
python3 -m http.server 8096 --bind 127.0.0.1
```

Open `http://127.0.0.1:8096/`.

## Files
- `index.html`, `styles.css`, `script.js` — prototype
- `public/assets/` — locally copied public website assets
- `ASSET-SOURCES.json` — exact source URLs and file sizes
- `DESIGN.md` — extracted design system and implementation rules
- `preview/` — desktop/mobile QA captures

## Important
This is a concept preview, not the live website. The inquiry form deliberately does not send anything. Confirm the business owner's permission for imagery, logo, copy, claims, testimonials, and awards before publishing or launching.
